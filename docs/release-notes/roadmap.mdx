ProBot ships in small, here's what's coming next.

## Upcoming Release

- **Multi-bot generation & management** - each user creates and manages N bots; the sidebar bot switcher and "Create New Bot" flow become fully load-bearing, including "create from a saved preset."
- **Dynamic thinking / generating messages** - richer, phase-aware status while the bot responds.
- **Portfolio website scraping** for the knowledge base.
- **Knowledge base as a draggable graph** - visualize and curate your sources.
- **Dashboard walkthrough modals** for new users.
- **Blog section** with a publishing pipeline.
- **Feedback button + upvote/downvote** on submitted feedback.
- **"Delete all knowledge base"** one-click reset.
