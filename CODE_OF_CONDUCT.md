# Code of Conduct

This project adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). The canonical text at that URL is the authoritative version and governs interactions in all project spaces — issues, pull requests, discussions, and any other community surface tied to this repository.

## Scope

This Code of Conduct applies within all project spaces, and also when an individual is officially representing the project in public spaces. Representation includes using an official project email address, posting via an official social-media account, or acting as an appointed representative at an online or offline event.

## Reporting

Report violations privately. Please **do not** open public issues or pull requests to describe an incident.

- **Email:** `vishal18@umd.edu`
- **Confidentiality:** All reports are treated confidentially. We share details only with the people necessary to investigate and respond.
- **Response time:** You should expect an acknowledgment within **72 hours** and a substantive follow-up within **7 days**.

## Enforcement

Project maintainers are responsible for clarifying and enforcing the standards described in the canonical Contributor Covenant text. Outcomes range from a private clarification through to a permanent ban from project spaces, applied at maintainer discretion and proportionate to the incident.
